# 🚀 ASTRA-KAVACH  
### 🛡️ Agentic AI OS for Safety, Security & Emergency Response

> **ASTRA-KAVACH** is a unified **Agentic AI Operating System** that protects both **physical environments and cyber spaces** by detecting threats, analyzing incidents, and triggering real-time alerts and emergency actions.

---

## 📌 Overview

ASTRA-KAVACH integrates **computer vision, cyber monitoring, voice AI, and real-time alerting** into one intelligent system. It acts as a **digital guardian**, capable of identifying danger, understanding situations, and coordinating rapid responses.

---

## 🎯 Key Features

### 🧍 Physical Safety & Threat Detection
- ESP32-CAM based real-time monitoring  
- Detects:
  - Weapons (guns, dangerous objects)
  - Suspicious activity
  - Accidents  
- Generates:
  - Scene description
  - Motion analysis  
- Triggers:
  - 🚨 Voice alerts (“Danger detected”)  
  - 📲 Telegram alerts with image & details  
  - 📍 GPS location sharing  

---

### 🚑 Emergency Response System
- Accident detection & automatic escalation  
- Finds nearest hospital (via API integration)  
- Sends GPS location & incident summary  
- Voice agent can:
  - Speak emergency message  
  - Assist in contacting authorities  

---

### 🌐 Cyber Security Monitoring
- Detects:
  - Wi-Fi attacks  
  - Bluetooth threats  
  - Evil Twin / SSID cloning  
  - DDoS indicators  
- MCP-connected AI workflows for analysis  
- Alerts via:
  - Voice agent  
  - Popup / wearable alerts  
  - Telegram bot  

---

### 📡 Hardware-Based Threat Detection
- Detects suspicious devices:
  - Wi-Fi jammer  
  - Bluetooth jammer  
  - Rogue RF activity (e.g., Flipper-like devices)  
- Real-time alerts via:
  - Voice  
  - Telegram  
  - Popups  

---

## 🧠 Agentic AI Architecture

ASTRA-KAVACH uses a **multi-agent system**:

- 🧠 Orchestrator Agent  
- 👁️ Vision Agent  
- 🛡️ Cyber Security Agent  
- 🚨 Emergency Agent  
- 🗣️ Voice Agent  
- 📲 Alert Agent  
- 📍 Location Agent

|----------------|-----------|

| AI Integration  | Claude (MCP), Vision Models |

| Alerts          | Telegram Bot API |
